VEIL DEMO

A simple cross-tab chat to test the Veil encryption extension. Pick a name, then open this page in another tab and pick a different name.

How to test Veil

  1. Install the Veil extension
  2. Open this page in Tab 1 — pick a name
  3. Open this page in Tab 2 — pick a different name
  4. Click the Veil icon → Set up for this site
  5. Type something in the input, then click Send
  6. Click the Veil icon → Start Session
  7. Do the same in the other tab — handshake completes automatically
  8. Send messages — they arrive encrypted and decrypt on the other side
