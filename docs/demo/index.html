Try Veil Encryption

See how end-to-end encryption works — two users, two tabs, real cryptography. No extension needed.

HOW THIS DEMO WORKS

  1. Open Bob and John in two separate tabs
  2. Click "Start Session" in one tab — the other will auto-connect
  3. Send messages — they're encrypted with AES-256-GCM in real time
  4. Refresh a tab — the keys are gone. The encrypted messages remain, but you can't read them anymore
  5. This is exactly what the messenger's server sees: unreadable ciphertext
